# SAGE BOOK - AI 智能创作平台

你的智能创作伙伴，用 AI 激发写作灵感。

## 🚀 快速开始

### 本地开发

```bash
# 直接打开 index.html 即可使用
open index.html
```

### 网页部署

```bash
# 使用 Python 简单启动服务器
python -m http.server 8000

# 或使用 Node.js
npx serve

# 然后访问 http://localhost:8000
```

## ✨ 核心功能

- **📝 主编辑器** - 全功能写作编辑器，支持 AI 续写、自动保存
- **👥 角色管理** - 创建和管理小说角色
- **🌍 世界观管理** - 地理、时间线、势力系统
- **📊 写作分析** - 节奏分析、伏笔追踪、风格洞察
- **🤖 AI 深度集成** - AI 辅助写作、写作教练
- **🕸️ 知识图谱** - 可视化故事结构
- **📚 模板库** - 12+ 写作模板
- **📋 大纲编辑器** - 树形大纲结构
- **⚙️ 设置** - 配置 AI 模型

## 🛠️ 技术栈

- HTML5
- CSS3 (CSS Variables)
- JavaScript (原生)
- LocalStorage (本地数据持久化)
